#  환율 알림 시스템 기획서 (Smart Rate Alert)

## 1. 개요 (Overview)
사용자가 설정한 특정 통화의 환율이 목표가에 도달했을 때 실시간 푸시 알림을 제공하는 기능입니다. 잦은 환율 확인의 번거로움을 해결하고 최적의 환전/투자 타이밍을 제공하는 것이 핵심입니다.
안드로이드 앱 알림으로 구현해줘. 

---

## 2. 상세 기능 및 UI 명세

### [화면 A] 알림 목록 (Alert Dashboard)
- **헤더**: `새 알림 추가` 플로팅 버튼 (+).
- **리스트 구성**:
  - **통화 정보**: 국기 아이콘 + 통화 코드 (예: 🇺🇸 USD).
  - **목표 정보**: 설정 가격 + 조건 (예: `1,310원 이하 시`).
  - **현재 상태 비주얼**: 현재가와 목표가 사이의 간격을 **% 게이지(ProgressBar)**로 시각화.
  - **제어**: 활성화/비활성화 토글 스위치, 왼쪽으로 슬라이드 시 삭제 버튼 노출.

### [화면 B] 알림 설정 (Setup Sheet)
- **통화 선택**: 주요 통화 리스트 및 검색 기능.
- **가격 설정**: 
  - 직접 입력 필드.
  - **퀵 버튼**: `-1%`, `-3%`, `-5%` (현재가 대비 자동 계산).
- **조건 설정**: `미만(하락 시 알림)` vs `초과(상승 시 알림)` 라디오 버튼.
- **옵션**: 알림 반복 여부 (1회성 / 조건 충족 시마다).

## 3. 기술 로직 (Technical Logic)

### 3.1. 데이터 스케줄링
- **수집 주기**: 환율이 업데이트되었을 때" 알림을 보내는 방식으로 구현해줘.
- **비교 로직**:
  - `Condition == 'Below'` : `Current_Rate <= Target_Rate` 일 때 트리거.
  - `Condition == 'Above'` : `Current_Rate >= Target_Rate` 일 때 트리거.
