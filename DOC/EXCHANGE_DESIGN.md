# 환율 계산기 디자인 명세 (Exchange Calculator Design Spec)

> 스크린샷 참조 앱: Converter Plus 스타일 기반 모바일 퍼스트 디자인

---

## 레이아웃 구조

```
┌─────────────────────────────────┐
│         [Header / AppBar]       │  ← 고정 (sticky)
├─────────────────────────────────┤
│         [StatusBar]             │  ← 오프라인 시에만 표시
├─────────────────────────────────┤
│                                 │
│         [CurrencyList]          │  ← 스크롤 가능 영역
│   (국기 + 코드 + 환산금액 행)   │
│                                 │
│         [+ 통화 추가 버튼]       │
│                                 │
├─────────────────────────────────┤
│         [NumericKeypad]         │  ← 고정 (sticky bottom)
└─────────────────────────────────┘
```

---

## 컴포넌트별 디자인 명세

---

### 1. Header (AppBar)

```
┌──────────────────────────────────────┐
│  ←   Exchange Calculator   🔍  ☰   │
└──────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 배경 | `bg-white` + `border-b border-slate-100` |
| 높이 | `h-14` (56px) |
| 타이틀 | `text-base font-semibold text-slate-900` |
| 아이콘 | 검색(🔍), 메뉴(☰) — `text-slate-600` |
| 레이아웃 | `flex items-center justify-between px-4` |

---

### 2. StatusBar (오프라인 배너)

```
┌──────────────────────────────────────┐
│ ⚠  OFFLINE MODE ACTIVE              │
│    LAST UPDATE: 2024.01.15 14:20    │
└──────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 배경 | `bg-amber-500` |
| 텍스트 | `text-white text-xs font-semibold tracking-wider uppercase` |
| 아이콘 | `⚠` 경고 아이콘 |
| 업데이트 시각 | `text-amber-100 text-xs` |
| 패딩 | `px-4 py-2` |
| 표시 조건 | `isOffline === true` 일 때만 렌더링 |

---

### 3. CurrencyRow (통화 행)

```
┌────────────────────────────────────────────┐
│  🇺🇸  [USD  ▼]              200.00        │  ← 일반 행
├────────────────────────────────────────────┤
│  🇰🇷  [KRW  ▼]          268,360.00  ✓   │  ← 기준통화 행 (파란색 강조)
└────────────────────────────────────────────┘
```

#### 일반 행 (Non-base)

| 속성 | 값 |
|------|-----|
| 배경 | `bg-white` |
| 높이 | `h-16` (64px) |
| 패딩 | `px-4` |
| 구분선 | `border-b border-slate-100` |
| 국기 크기 | `text-2xl` (이모지 플래그) |
| 통화 코드 | `text-sm font-semibold text-slate-800` |
| 드롭다운 | `text-slate-400 text-xs` (▼ 아이콘) |
| 금액 | `text-lg font-bold text-slate-900 tabular-nums` |
| 탭 동작 | 해당 행을 기준통화로 설정 |

#### 기준통화 행 (Base currency — 강조)

| 속성 | 값 |
|------|-----|
| 배경 | `bg-blue-50` |
| 좌측 보더 | `border-l-4 border-blue-500` |
| 통화 코드 | `text-blue-700 font-bold` |
| 금액 | `text-blue-900 font-extrabold` |
| 체크 아이콘 | `✓` — `text-blue-500 text-sm` (금액 우측) |

---

### 4. CurrencyList (통화 리스트)

| 속성 | 값 |
|------|-----|
| 오버플로우 | `overflow-y-auto` |
| 최대 높이 | `flex-1` (Header + StatusBar + Keypad 제외 나머지 전체) |
| 배경 | `bg-white` |
| 추가 버튼 | 리스트 하단, `+ 통화 추가` 텍스트, `text-blue-500` |
| 추가 버튼 배경 | `bg-slate-50 border-t border-dashed border-slate-300` |

---

### 5. NumericKeypad (숫자 키패드)

```
┌─────┬─────┬─────┬─────┬─────┬─────┐
│  C  │  ⇄  │  +  │  −  │  ×  │  ÷  │  ← 기능 행
├─────┼─────┼─────┼─────┼─────┼─────┤
│     │  7  │  8  │  9  │     │     │
├─────┼─────┼─────┼─────┤     │     │
│     │  4  │  5  │  6  │     │     │
├─────┼─────┼─────┼─────┤     │     │
│     │  1  │  2  │  3  │     │     │
├─────┼─────┼─────┼─────┤     │     │
│     │  .  │  0  │  ⌫  │     │     │
└─────┴─────┴─────┴─────┴─────┴─────┘
```

> 실제 레이아웃: 4열 기반 그리드, 기능키는 상단 2열 배치

#### 키패드 전체

| 속성 | 값 |
|------|-----|
| 배경 | `bg-slate-100` |
| 패딩 | `p-2` |
| 그리드 | `grid grid-cols-4 gap-1.5` |
| 위치 | 화면 하단 고정 (`sticky bottom-0`) |
| 그림자 | `shadow-[0_-2px_12px_rgba(0,0,0,0.08)]` |

#### 기능 키 (C, ⇄, +, −, ×, ÷)

| 속성 | 값 |
|------|-----|
| 배경 | `bg-white` |
| 텍스트 | `text-slate-600 font-semibold text-sm` |
| 모서리 | `rounded-xl` |
| 높이 | `h-12` |
| 탭 효과 | `active:bg-slate-200` |
| 특이사항 | `C` → `text-red-500`, `⇄` → `text-blue-500` |

#### 숫자 키 (0~9, .)

| 속성 | 값 |
|------|-----|
| 배경 | `bg-white` |
| 텍스트 | `text-slate-900 font-semibold text-xl` |
| 모서리 | `rounded-xl` |
| 높이 | `h-14` |
| 탭 효과 | `active:bg-slate-200` |

#### 백스페이스 키 (⌫)

| 속성 | 값 |
|------|-----|
| 배경 | `bg-white` |
| 텍스트 | `text-slate-500 text-xl` |
| 아이콘 | Lucide `Delete` 또는 유니코드 `⌫` |

### 6. CurrencySelector (통화 선택 모달)

```
┌─────────────────────────────────┐
│  통화 선택                    ✕  │
├─────────────────────────────────┤
│  🔍  국가 또는 통화코드 검색...  │
├─────────────────────────────────┤
│  🇺🇸  USD  미국 달러             │
│  🇰🇷  KRW  한국 원               │
│  🇬🇧  GBP  영국 파운드           │
│  ...                            │
└─────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 형식 | 바텀 시트 (Bottom Sheet) |
| 배경 | `bg-white rounded-t-3xl` |
| 오버레이 | `bg-black/40` |
| 검색 인풋 | `bg-slate-100 rounded-xl px-4 py-3 text-sm` |
| 통화 행 | 국기 이모지 + 코드(bold) + 이름(light) |
| 선택 강조 | `bg-blue-50 text-blue-700` |

## 반응형 가이드

| 기준 | 설명 |
|------|------|
| 기본 | 모바일 퍼스트, 최적 너비 `375px ~ 430px` |
| 최대 너비 | `max-w-sm mx-auto` (데스크탑에서도 모바일 UI 유지) |
| 키패드 높이 | 뷰포트 기준 `~35%` 차지 |
| 통화 리스트 | 나머지 공간 전체 (`flex-1 overflow-y-auto`) |

## 인터랙션 정의

| 동작 | 결과 |
|------|------|
| 통화 행 탭 | 해당 통화를 기준통화로 설정, 파란색 강조 |
| 숫자 입력 | 기준통화 금액 업데이트 → 전체 행 실시간 환산 |
| `C` 탭 | 입력 금액 초기화 (0) |
| `⌫` 탭 | 마지막 입력 문자 삭제 |
| `⇄` 탭 | 최상단 통화 ↔ 기준통화 스왑 |
| 통화 코드 탭 (▼) | 통화 선택 모달 열기 |
| `+ 통화 추가` 탭 | 통화 선택 모달 열기 (추가 모드) |
| 행 좌측 스와이프 | 삭제 버튼 노출 |
| 행 길게 누르기 | 드래그 재정렬 활성화 |

## 참조 스크린샷 분석

스크린샷 앱(Converter Plus) 기준 화면 구성:

- **Header**: 뒤로가기, 앱명, 검색, 메뉴 아이콘
- **StatusBar**: 주황 배너 — "OFFLINE MODE ACTIVE" / "LAST UPDATE: 2023.10.24 14:20"
- **통화 리스트**: USD, KRW(기준/파란색), GBP, CNY, JPY, RUB, VND 총 7개
- **기준통화(KRW)**: 배경 강조 + 체크 아이콘
- **키패드**: 상단 기능키 2개(C, ⇄) + 연산자(+−×÷) + 숫자 4×3 + 소수점 + 백스페이스
