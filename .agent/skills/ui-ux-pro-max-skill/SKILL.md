# UI/UX Pro Max Skill 🚀

이 스킬은 단순한 UI 생성을 넘어, 사용자 경험(UX)과 시각적 미학(UI)의 정수를 프로젝트에 반영하기 위한 지침과 데이터베이스를 제공합니다.

## 🎨 주요 구성 요소

### 1. UI 스타일 데이터베이스 (67+ Styles)
*   **Modern Styles**: Glassmorphism, Neumorphism, Claymorphism.
*   **Aesthetic Directions**: Minimalist, Brutalist, Retro-Futurism, Cyberpunk.
*   **Functional Styles**: Dashboard-centric, E-commerce optimized, SaaS-ready.

### 2. 프리미엄 컬러 팔레트 (96+ Palettes)
*   감성 기반 배색 (예: "Elegant Wellness", "Tech Noir", "Oceanic Calm").
*   WCAG AA/AAA 대비 표준 준수.
*   다크 모드 및 라이트 모드 간의 조화로운 변환 규칙.

### 3. 타이포그래피 페어링 (66+ Pairings)
*   가독성과 세련미를 모두 잡는 폰트 조합 제안.
*   Google Fonts 및 웹 표준 폰트 최적화 활용.

### 4. 고도화된 UX 지침 (96+ Rules)
*   **인지 부하 최소화**: 사용자가 고민하지 않고 행동하게 만드는 레이아웃 전략.
*   **피드백 루프**: 마이크로 인터랙션을 통한 상태 변화 알림.
*   **접근성 우선**: 모두가 사용할 수 있는 인터페이스 설계.

## 🛠 사용 방법

1.  **디자인 생성 시**: "UI UX Pro Max 스킬을 사용하여 [도메인명]에 어울리는 현대적인 디자인 시스템을 제안해줘"라고 요청하세요.
2.  **디자인 검토 시**: 작성된 코드를 분석하여 UX 가이드라인 위반 사항이나 개선 포인트를 찾아냅니다.
3.  **컴포넌트 개발 시**: 재사용 가능한 고품질 UI 컴포넌트의 사양을 정의합니다.

## 🎯 핵심 가치
*   **Anti-Generic**: 뻔한 AI 스타일을 탈피하고 브랜드 고유의 개성을 부여합니다.
*   **Production-Ready**: 즉시 서비스에 적용 가능한 수준의 정교한 코드를 지향합니다.
