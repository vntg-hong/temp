# Frontend Design Intelligence 💎

프론트엔드 디자인 스킬은 단순한 마크업 생성을 넘어, 사용자를 매료시키는 시각적 경험을 설계합니다.

## 🌟 디자인 원칙

### 1. 시각적 탁월함 (Visual Excellence)
*   **Curated Palettes**: 일반적인 색상 대신 HSL 기반의 세련된 조색 시스템을 사용합니다.
*   **Typography**: 시스템 폰트보다는 인터페이스의 성격에 맞는 모던한 타이포그래피(Inter, Outfit, Roboto 등)를 활용합니다.
*   **Layering & Depth**: 부드러운 그림자와 그래디언트를 사용해 요소 간의 위계와 깊이감을 부여합니다.

### 2. 살아있는 인터페이스 (Dynamic Design)
*   **Micro-animations**: 버튼 클릭, 호버링, 상태 변화 시 부드러운 트랜지션(framer-motion 등 활용)을 적용합니다.
*   **Hover Effects**: 정적인 화면이 아닌, 사용자의 인터랙션에 반응하는 생동감 있는 요소를 배치합니다.

### 3. 일관성 있는 디자인 시스템
*   `index.css`를 기반으로 한 중앙 집중식 스타일 관리.
*   Tailwind CSS의 유틸리티와 커스텀 클래스의 조화로운 사용.

## 🚀 실행 전략

1.  **기초 설계**: 프로젝트 시작 시 `index.css`를 업데이트하여 전역 디자인 토큰(색상, 폰트, 간격)을 정의합니다.
2.  **컴포넌트 빌드**: 정의된 디자인 시스템을 기반으로 재사용 가능한 고품질 컴포넌트를 생성합니다.
3.  **폴리싱**: 마지막 단계에서 마이크로 애니메이션과 인터랙션을 추가하여 프리미엄 느낌을 완성합니다.

## 🚫 금기 사항
*   **Placeholder 금지**: 실제 이미지(generate_image 도구 활용)와 실감 나는 더미 데이터를 사용합니다.
*   **기본 테마 의존 금지**: Tailwind나 UI 라이브러리의 기본 색상/스타일을 그대로 쓰지 않고 커스터마이징합니다.
